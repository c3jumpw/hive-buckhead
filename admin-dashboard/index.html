Cross-Page Sync Note
Currently, the three pages (index.html, floor.html, admin.html) share state via localStorage through the shared.js module. Changes in one tab broadcast to others via the storage event.

For production: Replace the load() / save() functions in shared.js with API calls to your ClickUp webhook endpoint. All three pages will then read/write from ClickUp as the single source of truth, with Google Sheets as the archive layer.
